什么是 POPO CLI？

一行命令安装，让你的 AI Agent 即装即用地调用 POPO 生态能力。 无论是发消息、创建文档、读写表格还是管理日程 —— Agent 装了 CLI，就像人装了 POPO 客户端。

🚀

3 步开始使用

1

安装 CLI

一行命令，安装 CLI 和全部 Skills

$ npm install -g @popo/cli →
2

选择 Agent

安装后选择要适配的 Agent，自动注入 Skills 文件

? 选择Agent安装目录
❯ CodeMaker
  Cursor
  Claude Code
  其他… →
3

OpenID 登录

自动弹出 OpenID 登录页，完成登录、授权即可使用

✓ 登录成功
✓ Skills 已就绪
✓ 开始使用 POPO CLI 🎉
