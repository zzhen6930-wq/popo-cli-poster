POPO CLI 正式上线

AI 时代，POPO 的新形态 —— 过去 POPO 服务人，现在 POPO CLI 服务 Agent

什么是 POPO CLI？

一行命令安装，让你的 AI Agent 即装即用地调用 POPO 生态能力。无论是发消息、创建文档、读写表格还是管理日程 —— Agent 装了 CLI，就像人装了 POPO 客户端。

仅需一行指令，即可在 Agent 操作 POPO

复制以下命令到终端运行即可完成安装

$ npm install -g @popo/cli
